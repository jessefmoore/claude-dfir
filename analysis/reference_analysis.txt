REPLAY ENGINE:
  — tick() called on requestAnimationFrame loop
  — replaySec advances by speed multiplier × real elapsed time
  — setScrubber(sec) seeks to a specific time
  — applyEventsForTime(sec) fires all events up to that time
  — currentEventIdx tracks current position
  — Speed multiplier: 1×/60×/240×/900× (default 60×)
  — Play/Pause toggles the tick loop
  — Reset clears all state (edge classes, host classes, currentEventIdx)

CHAPTER NAV:
  — IntersectionObserver with threshold 0.1 marks sections as active
  — Active section's nav link gets .active class
  — Toggle button: aria-expanded, class .open on .chapter-nav

HEATMAP:
  — JS generates heatmap-grid content (rows for hosts × columns for minutes)
  — Phase bars and key-event markers injected by JS
  — Cell data built from embedded HEATMAP_DATA object
  — Hover: shows .heatmap-tip with timestamp and event description
  — Cell click: calls setScrubber() to sync with replay

DRAWER:
  — DRAWER_DATA object maps host IDs to event lists
  — Host node click (SVG) or host card click opens drawer
  — Populates #drawerBody with formatted event rows

SCROLL FADE:
  — body.js-fade class added on DOMContentLoaded
  — IntersectionObserver adds .in class to .reveal elements when 10% visible

TTP LINKING:
  — .ttp-link and .eid-link clicks → scroll to + flash target element
  — .flash class added → flashHighlight animation → class removed after 1.8s

LEDGER FILTER:
  — Pills: set activeFam filter, toggle .active class
  — Search input: filter by textContent.includes(query)
  — Both apply() simultaneously; update #ledgerShowing count

COMMAND FLASH:
  — FLASH_EVENTS dict maps timestamp strings to flash config
  — On replay advance: if current event.t matches, call spawnFlash()
  — spawnFlash: creates .cmd-flash div, positions near source SVG node,
    appends to graph-wrap, removes after 2.4s
  — Impact events also spawn .impact-banner
  — svgToDOM() converts SVG viewBox coordinates to DOM pixels

PACKET ANIMATION:
  — PACKET_ROUTES dict maps event timestamps to { from, to, lane }
  — animatePacket(from, to, lane): moves .replay-packet SVG circle
    along a path from source node center to dest node center
    using linear interpolation over 800ms via requestAnimationFrame

HOST FOCUS (SVG):
  — Click on .host-clickable: collect connected edge-g elements
  — Add .host-focus / .edge-focus to related nodes/edges
  — Add .graph-focused to SVG element (triggers CSS dimming)
  — Click again or Esc: clear all focus classes
